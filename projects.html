A platform designed to help beginner programmers learn faster. Features an interactive code editor and AI assistant.

This platform was developed to help beginner programmers learn faster by providing a simple and interactive environment. The project features a real-time code editor and an AI assistant that helps with explanations and suggestions during the learning process. The application was built using only HTML, CSS, and JavaScript, focusing on performance, simplicity, and accessibility. One of the main challenges was creating an intuitive user experience without using frameworks, which was solved through a well-structured layout, efficient DOM manipulation, and real-time user feedback. The result is a lightweight and practical platform, ideal for those who are just starting in programming.
